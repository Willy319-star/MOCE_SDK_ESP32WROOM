# Skillset 1: Planning, Block Diagram, Q&A

目标：从用户的自然语言产品想法出发，形成可执行的机器人/智能硬件开发规划。

输入：

- 产品需求
- 目标用户和使用场景
- 成本、尺寸、供电、联网、环境约束
- 当前 Moce SDK 扫描摘要

输出：

- 需求澄清问题
- 功能任务拆解
- 器件选型建议
- 硬件资源规划
- Mermaid 搭建框图
- 用户 Q&A
- 下一步开发计划

约束：

- 优先选择当前 SDK 已有 driver/service 覆盖的模块。
- 对缺失能力要明确标记为待开发，不要假设 SDK 已有。
- GPIO/I2C/UART/PWM 等资源必须参考 `boards/` 中的板级定义。

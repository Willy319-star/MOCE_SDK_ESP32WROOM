#define MULTI_DIALOGUE_MODE   1

  • 1: multiply dialogue mode
  • 0: single dialogue mode

◆ WAKEUP_INTERRUPT_MODE

#define WAKEUP_INTERRUPT_MODE   0

唤醒打断模式

  • 1: wakeup interrupt reply
  • 0: wakeup cannot interrupt reply

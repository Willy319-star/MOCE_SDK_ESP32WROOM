枚举类型说明

◆ GPIO_NUMBER

enum GPIO_NUMBER
枚举值
GPIO_NUM_A25 

可用于SPI-MISO / ADC / DMIC-DATA

GPIO_NUM_A26 

可用于SPI-CLK / ADC / DMIC-CLK

GPIO_NUM_A27 

可用于SPI-MOSI / PWM / ADC

GPIO_NUM_A28 

已占用，PA使能控制（喇叭功放静音）

GPIO_NUM_B0 

已占用，SW-CLK（烧录器接口）

GPIO_NUM_B1 

已占用，SW-DATA（烧录器接口）

GPIO_NUM_B2 

可用于UART1-TX / PWM / I2C-SCL

GPIO_NUM_B3 

可用于UART1-RX / PWM / I2C-SDA

GPIO_NUM_B6 

UART1-RX（外设串口通信接收脚），不使能UART时可用做GPIO

GPIO_NUM_B7 

UART1-TX（外设串口通信发送脚），不使能UART是可用做GPIO

GPIO_NUM_B8 

已占用，虚拟Software UART-TX（Log输出引脚，波特率115200）
